1. 공유 URL 생성
ERD 데이터를 gzip + base64로 압축해 URL에 인코딩 — 서버 없이 링크 하나로 현재 상태를 공유. ?erd=xxxxx 파라미터로 열면 그대로 복원

2. JOIN 경로 탐색기
두 엔티티를 선택하면 FK 관계를 따라 연결 경로를 자동으로 찾아 A → B → C 형태로 표시하고 해당 JOIN SQL 자동 생성. "이 두 테이블 어떻게 연결하지?" 를 즉시 해결

3. 스냅샷 diff 뷰어
두 스냅샷을 선택해 변경된 엔티티/컬럼을 코드 리뷰처럼 +/- 하이라이트로 비교. 이미 diff 관련 CSS(.diff-added, .diff-removed)가 코드 안에 있어서 반쯤 준비된 상태

4. 타임라인 슬라이더
스냅샷 목록을 가로 타임라인으로 표시하고 슬라이더를 드래그하면 캔버스가 과거 상태로 실시간 전환 — "언제 이 컬럼이 생겼지?" 를 직관적으로 탐색

5. 정규화 진단
속성 이름 패턴을 분석해서 addr1, addr2, addr3 같은 반복 컬럼이나 복합 FK 없이 N:M 관계를 직접 연결한 경우를 감지해 노란색 경고 배지 표시

6. 데이터 볼륨 시각화
각 엔티티에 예상 행 수를 입력하면 엔티티 카드 크기나 테두리 두께로 규모를 표현 — 어느 테이블이 병목일지 설계 단계에서 바로 파악
