# UXERManager 메뉴별 감사 보고서 — 개요

## 요약

UXERManager 전체 25개 메뉴를 대상으로 함수 연결, 영속성 무결성, 단축키 동기화, 이스케이프/XSS 방어, 상태 수명 관리를 점검한 결과 총 57건의 오류가 발견되었다. 초기 자동 수정은 41건이었고, 이후 수동 검토로 분류됐던 항목과 메뉴 간 중복으로 stale 처리됐던 항목까지 후속 처리하여 **현재 49건이 수정 완료, 수동 검토 잔여 0건**이다(나머지는 검증 결과 '오류 아님'). 특히 DB 프로파일 관리의 onclick 이스케이프 결함 5건은 `_pmEscJsAttr` 헬퍼 도입으로 모두 해결했다. 대부분의 메뉴는 saveState/render/renderEntityTree 호출 흐름과 전체백업 통합이 잘 갖춰져 있어 전반적인 코드 건강도는 양호한 편이나, `toggleGridSnap`·`toggleNotation`·`setViewMode`의 saveState 누락으로 인한 영속성 위반이 여러 메뉴(05, 08, 24)에 걸쳐 반복되는 구조적 패턴이 확인되며, 인라인 onclick 핸들러의 이스케이프 미흡(03, 04, 13, 14, 19)과 타임라인 미리보기 중 단축키 격리 미흡(12), 공유 URL 복원 실패 시 기존 데이터 덮어쓰기(18), 다이어그램 전환 시 스테일 참조(22) 등 중요 결함이 잔존한다.
